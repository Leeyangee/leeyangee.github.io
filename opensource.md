# [](#header-1)leectf

leectf是一个小型的基于python3.6的包，其包含了多种CTF常见算法及数据可视化API，专注于提升CTF和实战的效率  

生成二阶希尔伯特曲线:  
```py
>>> import leectf
>>> leectf.algo.hilbert(2)
[(0, 0), (1, 0), (1, 1), (0, 1), (0, 2), (0, 3), (1, 3), (1, 2), (2, 2), (2, 3), (3, 3), (3, 2), (3, 1), (2, 1), (2, 0), (3, 0)]
```
